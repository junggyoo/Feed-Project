# [코멘토 피드 시스템 프로젝트]

## Introduction
 - "인재와 기업의 꿈을 연결하는 코멘토"의 피드 시스템을 구현하였습니다. 

## Project Period & Members
 - 개발 기간 : 2020/10/6 ~ 2020/10/9
 - 개발 인원 : 1명 

## Features
 - 반응형 웹 (모바일/테블릿 768px 이하)
 - 무한로딩(스크롤) 
 - 피드 정렬 기능(오름차순, 내림차순)
 - 필터 모달(카테고리별)
 - 동적 라우팅 디테일 페이지 이동
 - Vuex 상태 관리 

## Technologies
 - Vue.js
 - Vue-router
 - Vuex
 - SCSS
 - JavaScript(es6)
 - Git
 - Axios
